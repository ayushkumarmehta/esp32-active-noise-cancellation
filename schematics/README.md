# Schematics

This folder contains circuit diagrams for the ANC system.

## Files
- `anc_circuit_diagram.png` — Full circuit connection diagram
- `mic_amplifier_stage.png` — Microphone amplifier front-end

## How to Add Your Circuit Diagram
1. Draw your schematic in **LTspice** or **Fritzing** (free tools)
2. Export as PNG
3. Upload here by clicking "Add file" → "Upload files"

> Circuit diagram will be added once hardware prototype is finalised.
